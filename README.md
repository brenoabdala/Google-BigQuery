# Google BigQuery - Criando Estrutura a estrutura do Banco de Dados

<p>Este repositórtio tem como objetivo, mostrar o principais mostrar a estruturação inicial do BigQuery.</p>

<p><b>Acesse o portal:</b> https://console.cloud.google.com;</p>
<p>No menu lateral seleciona a opção <strong>BigQuery</strong>;</p>

 
<hr>
